This dashboard simulates how three actors — a Designer, a Smartphone, and a User — interact over time using Active Inference: a cognitive model where each agent continuously updates its beliefs about the world and chooses actions to minimise surprise. Adjust the parameters below and press Run to observe how each agent's behaviour evolves.

Agent 1
The Designer
Sets global defaults for the smartphone: how many notifications to send by default, and how hard it is for the user to turn them off.
Goal: maximise engagement metrics (business objective)
Agent 2
The Smartphone
Controls the interface the user actually sees: notification frequency, content ranking, and recommendation aggressiveness.
Goal: maximise immediate engagement, minimise churn
Agent 3
The User
Responds to the interface by choosing to engage, ignore, change settings, activate Do Not Disturb, or uninstall. Starts with no clear preference — learns over time.
Goal: task completion, low interruption, positive mood
