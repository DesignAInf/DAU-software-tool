This dashboard studies how Designer and Smartphone strategies differentially manipulate users with different psychological profiles. Four user archetypes — Achiever, Social, Anxious, Resistant — run in parallel under the same environment, revealing which persuasive techniques are most effective on which profiles.

Profile 1
The Achiever
Productivity-oriented, streak lover. Values task completion and time well spent above all.
Vulnerable to: reward loops, streak pressure
Profile 2
The Social
Validation-seeking, FOMO-prone. Highly reactive to social signals and comparison.
Vulnerable to: social amplification, peak targeting
Profile 3
The Anxious
High baseline stress, uses phone as escape. Seeks mood improvement above all.
Vulnerable to: variable reward, peak targeting
Profile 4
The Resistant
High media literacy, aware of manipulation. Strongly averse to wasted time and annoying notifications.
Responds positively to: transparency, ethical nudging
