# DAU model — rigid designer (pymdp)

Multi-agent Active Inference demo with a rigid designer. Depends on **pymdp** (`inferactively-pymdp` on PyPI).

## Dependencies

```bash
pip install -r requirements.txt
```

## Run

From this directory:

```bash
python "dau_active_inference(5).py"
```

Use script arguments as documented in the source file.

For additional context see `README (1).md` if present.
