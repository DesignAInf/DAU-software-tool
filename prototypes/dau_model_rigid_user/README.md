# DAU model — rigid user (pymdp)

Multi-agent Active Inference demo with a rigid user policy. Depends on **pymdp** (`inferactively-pymdp` on PyPI).

## Dependencies

```bash
pip install -r requirements.txt
```

## Run

From this directory:

```bash
python dau_active_inference.py
```

Use script arguments as documented in the source file (`argparse`).

For additional context see `README (1).md` if present.
